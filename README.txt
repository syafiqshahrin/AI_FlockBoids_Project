Boids Simulation Controls:
- W A S D to move attractor object (blue diamond)
- Left Click inside window to add flock object (green triangle)
- Right Click inside window to add obstacle object (white circle)
- Press C to clear all flock objects and obstacle objects from the scene

Additional Details:
- Predator object is rendered as a red triangle
- Predators will move autonomously
